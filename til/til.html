HTML Tag

No

Nama Tag

Fungsi

1

<html>... </html>

Tag pembuka untuk membuat dokumen HTML

2

<head>...</head>

Informasi meta tentang dokumen

3

<title>...</title>

Membuat judul yang nantinya ditampilkan di browser
4

<body>...</body>

Tempat dibuatnya semua konten website menggunakan HTML
5

<h1>...</h1> - <h6>...</h6>

Tempat dibuatnya semua konten website menggunakan HTML,yang membedakan antara h1 dan h6 adalah ukuran font,bilamana h1 lebih besar daripada h6

Contoh penggunakan HTML Tag

  • Pertama tama kita membuat file dengan extension .html ,contoh : index.html

  • Kemudian kita masukkan code seperti dibawah ini

<!DOCTYPE html>

<html>

<head>

<title>Tes123</title>

</head>

<body>

<h1>Tes123</h1>

</body>

</html>

  • Kemudian kita save,lanjut cek ke browser teman teman
  • apabila kita sudah cek di browser dengan hasil yang samaseperti diatas berarti kita sudah sukses
